# 💰 Smart Expense Tracker

A professional Python-based Expense Tracker application built with SQLite and Matplotlib for tracking daily expenses, analyzing spending habits, and visualizing financial data.

---

## 🚀 Features

- ➕ Add daily expenses
- 📂 Categorize expenses (Food, Travel, Shopping, etc.)
- 📊 View all expenses
- 📈 Generate category-wise expense charts
- 💾 SQLite database integration
- 🖥️ Simple command-line interface
- 🔍 Financial insights through visualization

---

## 🛠️ Tech Stack

- Python 3
- SQLite3
- Matplotlib
